Códigos de Producto por Proveedor
¿Para qué sirve esto?

Cada proveedor identifica tus productos con su propio código interno (generalmente distinto al código que vos usás en el sistema). Acá guardás esa relación, para que después puedas buscar productos por el código del proveedor al cargar una factura de compra.

Ejemplo: tu sistema tiene al producto con código 3293 pero el proveedor te lo factura como LON-12. Guardando esa relación, la próxima vez que cargues una factura de ese proveedor podés escribir LON-12 en el buscador y te encuentra el producto.

Cómo usar esta pantalla
  1. Seleccioná un proveedor del dropdown de arriba.
  2. Se carga la lista de todos los productos que ya tienen código asignado para ese proveedor.
  3. Acciones disponibles por fila:
    • Editar: cambiar el código del proveedor.
    • Eliminar (solo admin): quita la asociación.
  4. Botón "Agregar código de producto": abre un modal donde buscás el producto (por código interno o nombre) y asignás el código que usa el proveedor.
Otras formas de cargar códigos

Además de esta pantalla, los códigos del proveedor también se cargan automáticamente:

  • Al cargar una Nueva Factura con Detalle, en el campo "Cód. Proveedor" al lado de cada artículo.
  • En el Alta Rápida de Producto, si agregás un producto nuevo mientras cargás una factura.
Tené en cuenta
  • Cada producto puede tener un código distinto por cada proveedor.
  • Si cargás un código nuevo para una combinación que ya existía, se actualiza (no se duplica).
  • Si en el listado aparece "(inactivo)" es porque el producto fue dado de baja en el ABM de productos.
