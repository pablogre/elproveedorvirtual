Categorías de Productos
¿Para qué sirve esto?

Las categorías te permiten clasificar los productos (ej: fiambres, quesos, snacks) para organizar el ABM, aplicar filtros y generar reportes.

Cómo funciona
  • Crear nueva categoría: botón "Agregar categoría". Se guarda SIEMPRE en minúsculas para mantener consistencia.
  • Editar: cambia nombre, color y orden. Si cambiás el nombre, automáticamente se actualizan todos los productos que la usaban.
  • Eliminar (solo admin): solo se puede eliminar si NO hay productos usándola. Si quedaron productos, primero movelos a otra categoría.
  • Desactivar: alternativa a eliminar. La categoría desaparece del dropdown pero los productos siguen en esa categoría.
Tip

Usá el color para visualizar categorías rápido en el ABM de productos (ej: azul para fiambres, verde para verduras, etc.). Es opcional pero muy útil cuando tenés muchos productos.
